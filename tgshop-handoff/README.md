# TGShop Chat — Лендинг (UZ) — Handoff пакет для разработчика

Этот пакет содержит всё необходимое для реализации лендинга TGShop Chat и деплоя на ваш домен.

## Содержимое

| Документ | Что внутри |
|---|---|
| [`TZ.md`](./TZ.md) | Главное техзадание: стек, структура, SEO, чек-лист |
| [`content-and-links.md`](./content-and-links.md) | Все тексты + URL для гиперссылок |
| [`design-tokens.md`](./design-tokens.md) | CSS-переменные: цвета, шрифты, отступы, тени |
| [`assets-checklist.md`](./assets-checklist.md) | Список ассетов для экспорта из Figma |
| `index.html` + `style.css` | Базовый HTML/CSS скелет (если приложен) |

## Быстрый старт

1. **Открыть Figma:** `https://www.figma.com/design/dW5U2kQyvtzYYgIiDuWSbZ/TGShop-Merchant`
   - Доступ: запросить у Виктории (CMO)
   - 3 фрейма-макета: Desktop 1440px, Tablet 1024px, Mobile 375px (страница «Vika Marketing»)
2. **Прочитать `TZ.md`** — общая картина
3. **Использовать `design-tokens.md`** — настроить CSS-переменные
4. **Сверстать секции по `content-and-links.md`** — все тексты и URL под рукой
5. **Экспортировать ассеты** по `assets-checklist.md`
6. **Проверить на брейкпоинтах** 375 / 768 / 1024 / 1440 px

## Срок и приоритет

- **Дедлайн:** «Завтра» от 2026-05-26 (по согласованию с Викторией)
- **Приоритет:** залить функциональную статическую версию на домен
- **Можно итерировать** анимациями и микро-полишем после первого деплоя
